# Fitness Salonu Yönetim Sistemi – Veritabanı

MySQL ile tasarlanmış fitness salonu yönetim sistemi veritabanı projesi.

## İçerik
- 15 ilişkisel tablo
- 5 view, 3 stored procedure, 4 trigger, 2 fonksiyon
- 10 performans indexi
- BMI hesaplama, üyelik takibi, antrenman logları

## Teknoloji
MySQL 8.0
